Detailed Description

A class the holds an audio player. To start playing a file call loadNewFile(). If a file is paused call playFromPause() rather than loadNewFile().

Loads and plays a new file

Parameters
fileToLoadthe file to load and play. If this is nullptr no file will be loaded.
playPointthe point in the file to start playing from in seconds. If this is blank transport will start from the start of the file.
stopPointthe point in the file to stop playing at in seconds. If this is blank transport will end at the end of the file.
sendNotificationAtEndwhether to send a notification in the form of an ActionMessage at the end of playback. This message is "Transport Finished".
